# تشغيل Image QR Code على Hostinger

الميزة الجديدة تستخدم PHP وSQLite لتخزين الصور وروابطها، بينما تبقى أدوات PDF وQR العادية داخل المتصفح.

## المتطلبات

- PHP 8.1 أو أحدث.
- تفعيل إضافتي `PDO_SQLite` و`fileinfo`.
- صلاحية كتابة لمجلد `storage` (عادة 750 أو 755 حسب إعدادات الاستضافة).
- شهادة SSL فعالة حتى تكون الروابط بصيغة HTTPS.

## خطوات النشر

1. ارفع كل محتويات مجلد `dist` مباشرة إلى `public_html`.
2. تأكد من وجود المجلدات `api` و`storage/images` والملف `.htaccess`.
3. من Hostinger افتح **Advanced > PHP Configuration** واختر PHP 8.1 أو أحدث.
4. اختبر الصفحة: `https://your-domain.com/image-qr-code`.
5. ارفع صورة تجريبية، ثم امسح QR Code من هاتف آخر.

قاعدة SQLite والسر السري للحماية يُنشآن تلقائيًا داخل `storage`. يمنع `.htaccess` الوصول المباشر إلى هذا المجلد، وتُعرض الصور فقط من خلال API بعد التحقق من الانتهاء وكلمة المرور.
