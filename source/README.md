# PDF & QR Tools

موقع ويب متجاوب بثلاث لغات: العربية، الألمانية، والإنجليزية.

## المميزات

- محرر PDF داخل المتصفح: نصوص، تعديل بصري للنص الموجود، رسم حر، تمييز، أشكال، أسهم، تراجع وإعادة، صفحات متعددة، تكبير وتنزيل.
- إدارة صفحات PDF: إضافة صفحة فارغة، حذف صفحة، دمج ملفات، وتقسيم أو استخراج نطاق صفحات.
- مولّد QR للروابط الذكية وأرقام الهاتف والنصوص والبريد الإلكتروني وواتساب.
- تخصيص ألوان QR وتنزيله بصيغتي PNG وSVG.
- صفحات رئيسية، خصوصية، وتواصل.
- اتجاه RTL كامل للعربية وتصميم متجاوب للموبايل.
- لا يتم إرسال ملفات PDF إلى خادم أو تخزينها.

## التشغيل

```powershell
pnpm install
pnpm dev
```

## تجهيز نسخة النشر

```powershell
pnpm build
```

ستجد النسخة النهائية الجاهزة للرفع على أي استضافة ثابتة داخل مجلد `dist`.

راجع [HOSTINGER_DEPLOY_AR.md](./HOSTINGER_DEPLOY_AR.md) لخطوات الرفع على Hostinger.
