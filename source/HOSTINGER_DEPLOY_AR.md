# رفع PDF & QR Tools على Hostinger

الموقع يعمل كاستضافة ثابتة ولا يحتاج قاعدة بيانات أو خادم Node.js. جميع عمليات PDF وQR تتم داخل متصفح المستخدم.

## الملفات الجاهزة

بعد تنفيذ:

```powershell
pnpm build
```

توجد نسخة النشر داخل مجلد `dist`.

## خطوات الرفع

1. افتح لوحة Hostinger ثم **Websites** واختر موقعك.
2. افتح **File Manager** ثم مجلد `public_html`.
3. احذف صفحة Hostinger الافتراضية فقط إن كانت موجودة.
4. ارفع **محتويات** مجلد `dist` مباشرة إلى `public_html`، وليس مجلد `dist` نفسه.
5. تأكد أن `index.html` و`.htaccess` ومجلد `assets` موجودون مباشرة داخل `public_html`.
6. افتح النطاق وجرّب محرر PDF وإنشاء QR.

## ملاحظات

- لا تحتاج إعداد قاعدة بيانات MySQL.
- لا تحتاج تشغيل `npm` أو `pnpm` على الاستضافة.
- ملف `.htaccess` يضيف نوع ملفات JavaScript الحديثة، ضغط الملفات، وسياسات أمان وتخزين مؤقت.
- يستخدم الموقع روابط داخلية تبدأ بـ `#/`، لذلك تعمل الصفحات مباشرة دون إعداد Rewrite إضافي.
- عند تحديث الموقع، ابنِ المشروع مرة أخرى وارفع محتويات `dist` الجديدة.
